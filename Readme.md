# 🏢 B2B TenderHub

**B2B TenderHub** is a full-stack B2B tender management platform that allows companies to register, publish tenders, browse and apply to other tenders, and discover potential B2B partners by industry or services.


## 📌 Features

- 🏢 Company Registration & Login
- 🧾 Tender Creation & Publication
- 🔍 Browse and Apply to Open Tenders
- 🧭 Company Search by Name, Industry, or Products/Services
- 📁 Company Profile with Logo and Service Info
- ☁️ File Upload using Cloudinary (for tender documents and logos)
- 💾 MongoDB for Data Storage
- 🔐 Authentication via JWT (Login / Signup)

---

## 🧱 Tech Stack

**Frontend:**
- React.js
- Bootstrap / Tailwind CSS
- Axios
- React Router

**Backend:**
- Node.js
- Express.js
- MongoDB & Mongoose
- Cloudinary for file uploads
- JWT for authentication
- Multer (for file handling)
